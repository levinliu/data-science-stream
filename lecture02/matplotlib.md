# Matplotlib 分享

## 安装后快速使用
```sh
python demo.py
```

## 定制并绘制一条折线图
```sh
python demo1.py
```
## 绘制多条折线图
```sh
python demoN.py
```

## 绘制多款散点图
```sh
python demo_scatter.py
```

## 切换backend
```sh
python demo_bkend.py
```
